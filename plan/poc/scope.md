# POC Scope

## Goal
Define the scope for the Egypt Edututor POC.

## In Scope
- Basic chat interface
- Content ingestion pipeline
- User authentication
- Local deployment with Docker Compose
- Knowledge graph visualization

## Out of Scope
- Production-grade infrastructure
- Advanced analytics
- Payment integration
- Mobile app
